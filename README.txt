-Création d'un site web pour une agence de voyage fictive de Nouvelle-Calédonie, que j'ai nommé Travel n'sea, afin d'avoir un jeu de mot intéressant entre "see" ( voir ) et "sea" ( la mer ).

-Crée "from scratch" uniquement avec HTML et CSS, sans Javascript.

-Le site est pour le moment statique mais il aurait vocation à devenir dynamique.

-Les icones et images proviennent de la banque d'image gratuite unsplash.com, un gros coup de coeur et libre de droit.
